# Minimal Web AI Assistant Interface

A beautifully crafted, ultra-lightweight, and futuristic web-based user interface for interacting with Large Language Models (LLMs) powered by the **Groq Cloud API**. This client is built entirely with vanilla HTML, CSS, and modern JavaScript, making it completely serverless and instantaneous to load.

## ⚡ Key Features

- **Futuristic & Clean UI:** Designed with premium typography (`Syne` and `DM Mono`) and a cyber-dark aesthetic.
- **Serverless & Private:** Connects directly from your browser to the Groq API endpoint. No database or external servers involved.
- **Local Storage Configurations:** Your API Key, selected model, username, and custom System Prompt are securely saved locally inside your browser's `localStorage`.
- **Dynamic Content Flow:** Includes smooth CSS animations, an auto-resizing text area, a native typing indicator, and responsive grid layouts.
- **Context Management:** Maintains a chat session history locally to provide coherent dialogue threads.
- **Prompt Suggestions:** One-click shortcut cards for quick testing and onboarding.

## 🛠️ Built With

- **Frontend Structure:** HTML5
- **Styling Architecture:** CSS3 (Featuring Custom CSS Variables, Flexbox, and CSS Grid)
- **Engine Control:** Vanilla JavaScript (ES6+, Fetch API, Async/Await)
- **AI Processing Backend:** Groq Cloud API Engine
